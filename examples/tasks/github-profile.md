# GitHub プロフィール README を作成・更新

## 目的

`README.md` を、GitHub プロフィールページ（`https://github.com/<username>`）向けに整える。

## 要件

- 日本語メイン（見出しは英語でも可）
- 自己紹介（開発者としての方向性・興味分野）
- 主要リポジトリへのリンク（例: CodePrism）
- 使用技術スタック（バッジ形式が望ましい）
- 連絡先または SNS（未設定ならプレースホルダーで明示）
- 変更は **README.md のみ**（プロフィール専用リポジトリ想定）

## 3 エージェントへの観点

- **Melchior**: リンク切れ・誤情報・要件漏れがないか
- **Balthasar**: 読みやすく、将来の更新がしやすい Markdown か
- **Caspar**: 画像・バッジを抑え、軽量で表示が速いか
